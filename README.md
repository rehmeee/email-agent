# Email Agent (Next.js + LangGraph)

Starter Next.js app for building an email assistant with LangGraph and OpenRouter.

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Environment

Your `.env` file is already in place with OpenRouter keys. Server-only secrets stay in `.env` (not committed).

## Planned features

- Google sign-in
- LangGraph agent with email tools
- Chat UI
